**建设用地规划许可证办理流程**

```
┌─────────────────────────────────────────────────────────────────────┐
│                    建设用地规划许可证办理流程                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  第一步：材料准备                                                     │
│  ├── 签订国有土地使用权出让合同                                        │
│  ├── 建设项目批准文件                                                 │
│  └── 土地出让合同等材料                                               │
│                                                                     │
│  第二步：窗口申报                                                     │
│  ├── 周一至周五工作日                                                 │
│  ├── 前往规资局窗口提交材料                                            │
│  └── 或通过天津网上办事大厅提交电子文件                                 │
│                                                                     │
│  第三步：材料审核                                                     │
│  ├── 窗口工作人员核收申报材料                                          │
│  ├── 如有错误当场更正                                                 │
│  └── 如材料不齐全当场告知需补正内容                                     │
│                                                                     │
│  第四步：项目审核                                                     │
│  ├── 项目经办人审核材料                                               │
│  ├── 需要补正的及时告知申请人                                          │
│  └── 审核合格后安排现场勘察                                            │
│                                                                     │
│  第五步：现场勘察                                                     │
│  ├── 项目经办人前往现场查看                                            │
│  └── 符合要求签署工作证明                                             │
│                                                                     │
│  第六步：领取许可证                                                   │
│  ├── 现场勘察合格后通知申请人                                          │
│  └── 到窗口领取建设用地规划许可证                                       │
│                                                                     │
│  关键科室：自然资源所有者权益科                                         │
│  办理时限：15个工作日                                                  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```
