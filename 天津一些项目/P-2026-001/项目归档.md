## 一、项目概述

### 1.1 项目背景

**时间窗口期**：中央预算内资金有一批支持方向已明确，但该城市报上来的项目太少，资金还有富余。

**核心约束**：
- 老旧小区内部已经申请过中央资金进行改造，不能重复申报
- 老旧小区与城市主干道之间的连接道路需要提升改造

**机遇识别**：
- 中央预算内资金主要投向老旧小区改造及周边连接道路
- 区管道路有三年或五年的工作计划，是已经批准过的
- 道路项目前期工作相对简单，能够快速立项

### 1.3 政策依据

**支持政策**：中央预算内投资专项管理办法

**支持范围**：
- 城镇老旧小区改造
- 小区外与城市主干网衔接的道路和公共交通、通信、供电等城镇基础设施配套项目

**政策契合度**：
- ✅ 精准落在"老旧小区周边配套"支持范围内
- ✅ 避免重复投资（小区内已改造，现在补短板形成闭环）
- ✅ 落地快、示范性强

## 二、项目内容

### 2.1 建设内容（三层结构）

#### 地上工程

**道路路面修复**：
- 裂缝处理（灌缝、贴缝、微表处）
- 坑槽修复（冷补、热补、嵌缝）
- 沉陷调整（注浆、换填、压实）
- 罩面修复（4cm细粒式沥青混凝土）
- 面层挖补（次干路：4cm+8cm；支路：4cm+6cm）
- 基层补强（一步/两步补强结构）

**人行道维修**：
- 花砖调整（重做3cm水泥砂浆垫层）
- 换砖（原有花砖形式+3cm水泥砂浆垫层）
- 翻修（重新施作基层+面层更换）

**附属设施**：
- 侧缘石及树穴石调整/更换
- 无障碍坡道与盲道完善
- 路灯升级
- 街角口袋公园建设
- 更换垃圾桶
- 增设充电桩

#### 地下工程
- 雨污分流管网更新
- 老化供水、排水、燃气、供热管网更新
- 避免未来"拉链路"现象

#### 路边空间
- 道路与小区围墙之间的"灰空间"利用
- 便民停车点设置
- 智慧灯杆接口预留

### 2.2 道路路面病害类型

| 病害类型 | 说明 |
|---------|------|
| 纵横裂缝 | 缝宽10mm及以内 |
| 网裂、龟裂 | 表面或深层 |
| 车辙 | 15mm以下 |
| 井周沉陷 | 局部病害 |
| 坑槽 | 零星分布 |
| 剥落、叽浆 | 基层病害 |

### 2.3 修复工艺选择

| 病害程度 | 修复方案 | 结构形式 |
|---------|---------|---------|
| 轻微裂缝（<10mm） | 罩面修复 | 4cm细粒式沥青混凝土 |
| 连续裂缝、网裂、龟裂 | 面层挖补 | 次干路：4cm+8cm；支路：4cm+6cm |
| 剥落、叽浆、坑槽、沉陷 | 基层补强 | 一步：+9cm ATB-30；两步：+18cm石灰粉煤灰碎石+18cm ATB-30 |

### 2.4 筛选标准

1. **符合老旧小区周边的道路**：市发改委明确要求
2. **剔除次干道**：次干道由市区级财政出资
3. **避免重复申报**：剔除已纳入城市更新项目的道路
4. **依托既有计划**：从区管道路既有计划中快速梳理
5. **不改变原道路宽度、标高**：仅对路面病害进行修复
