## 一、项目概述

### 1.1 项目背景

**时间窗口期**：中央预算内资金有一批支持方向已明确，但该城市报上来的项目太少，资金还有富余。

**核心约束**：
- 老旧小区内部已经申请过中央资金进行改造，不能重复申报
- 老旧小区与城市主干道之间的连接道路需要提升改造

**机遇识别**：
- 中央预算内资金主要投向老旧小区改造及周边连接道路
- 区管道路有三年或五年的工作计划，是已经批准过的
- 道路项目前期工作相对简单，能够快速立项

**支持政策**：中央预算内投资专项管理办法

**支持范围**：
- 城镇老旧小区改造
- 小区外与城市主干网衔接的道路和公共交通、通信、供电等城镇基础设施配套项目

## 二、项目内容

### 2.1 建设内容（三层结构）

#### 地上工程

**道路路面修复**：
- 裂缝处理（灌缝、贴缝、微表处）
- 坑槽修复（冷补、热补、嵌缝）
- 沉陷调整（注浆、换填、压实）
- 罩面修复（4cm细粒式沥青混凝土）
- 面层挖补（次干路：4cm+8cm；支路：4cm+6cm）
- 基层补强（一步/两步补强结构）

**人行道维修**：
- 花砖调整（重做3cm水泥砂浆垫层）
- 换砖（原有花砖形式+3cm水泥砂浆垫层）
- 翻修（重新施作基层+面层更换）

**附属设施**：
- 侧缘石及树穴石调整/更换
- 无障碍坡道与盲道完善
- 路灯升级
- 街角口袋公园建设
- 更换垃圾桶
- 增设充电桩

#### 地下工程
- 雨污分流管网更新
- 老化供水、排水、燃气、供热管网更新
- 避免未来"拉链路"现象

#### 路边空间
- 道路与小区围墙之间的"灰空间"利用
- 便民停车点设置
- 智慧灯杆接口预留

### 2.2 道路路面病害类型

| 病害类型 | 说明 |
|---------|------|
| 纵横裂缝 | 缝宽10mm及以内 |
| 网裂、龟裂 | 表面或深层 |
| 车辙 | 15mm以下 |
| 井周沉陷 | 局部病害 |
| 坑槽 | 零星分布 |
| 剥落、叽浆 | 基层病害 |

### 2.3 修复工艺选择

| 病害程度 | 修复方案 | 结构形式 |
|---------|---------|---------|
| 轻微裂缝（<10mm） | 罩面修复 | 4cm细粒式沥青混凝土 |
| 连续裂缝、网裂、龟裂 | 面层挖补 | 次干路：4cm+8cm；支路：4cm+6cm |
| 剥落、叽浆、坑槽、沉陷 | 基层补强 | 一步：+9cm ATB-30；两步：+18cm石灰粉煤灰碎石+18cm ATB-30 |

### 2.4 筛选标准

1. **符合老旧小区周边的道路**：市发改委明确要求
2. **剔除次干道**：次干道由市区级财政出资
3. **避免重复申报**：剔除已纳入城市更新项目的道路
4. **依托既有计划**：从区管道路既有计划中快速梳理
5. **不改变原道路宽度、标高**：仅对路面病害进行修复

## 五、项目意义

### 5.1 近期目标
- 拿到中央资金，快速启动
- 解决居民出行痛点
- 提升道路通行条件

### 5.2 中期目标
- 通过道路改善，带动沿街底商价值
- 盘活边角地（停车、广告等运营收益）
- 优化营商环境

### 5.3 远期目标
- 把路网作为骨架，串联起老旧街区、老旧厂房的更新项目
- 形成"片区滚动开发"态势
- 为整个城市更新打开一个口子

## 六、关键经验

### 6.1 成功要素

| 要素 | 说明 |
|------|------|
| **精准卡位** | 精准落在政策支持范围内 |
| **快速响应** | 一天之内跑通住建和城管系统 |
| **避免重复** | 提前剔除已改造的小区，形成闭环 |
| **系统思维** | 不是简单的"修路"，而是"连接器"和"片区更新"的切口 |

### 6.2 可复用经验

- 区管道路既有计划是快速出量的关键依据
- 道路改造不是孤立的，是"连接器"
- 从"工程"到"片区更新"的思路

### 6.3 风险提示

| 风险 | 说明 | 应对 |
|------|------|------|
| 背街小巷 | 属于治理灰色地带，没有明确的资金支持 | 暂不纳入，后续跟踪 |
| 体检要求 | 道路体检费用需要明确资金来源 | 纳入前期费或区里垫付 |
| 区财政约束 | 如果没有国家政策支持，区管道路要靠本级财政 | 积极争取中央资金 |

## 七、攻防问答要点

### 7.1 常见质疑与应对

| 质疑 | 应对策略 |
|------|---------|
| 时间紧如何保证质量？ | 依托既有计划，红头文件支撑 |
| 道路是否真正"衔接主干网"？ | 地图展示+服务导向论证 |
| 红头文件能否证明符合政策？ | 叠加分析，小区范围说话 |
| 混合小区如何证明专款专用？ | 片区生态逻辑，老旧小区居民占比 |
| 次干道为何剔除？ | 财政责任+地图论证 |
| 是否为"面子工程"？ | 安全出行+无障碍设施+管网配套 |
| 规划文件是否过时？ | 五年计划在有效期内 |
| 管网问题谁负责？ | 道路体检先行，摸排机制 |
| 区里省钱是否合理？ | 财政再投入，中央地方合力 |

## 八、关联项目

### 8.1 与背街小巷项目的关系

**道路来源**：本项目53条道路来源于背街小巷台账中的**区管道路（支路）**

**筛选过程**：
```
背街小巷统计（2024年冬）
    │
    ├── 区管道路（支路）──→ 有完整台账 ──→ 选定70条
    │                           ↓
    │                      河西项目申报
    │                           ↓
    │                      缩减17条（市发改委整合）
    │                           ↓
    │                      最终53条（20+33）
    │
    ├── 街坊路 ──→ 数据不完整，未申报
    │
    └── 连接路 ──→ 数据不完整，未申报
```

**申报主题变化**：
| 阶段 | 主题 | 说明 |
|------|------|------|
| 背街小巷工作 | 精细化治理 | 城管委系统工作 |
| 本项目申报 | **道路+管网** | 适配中央预算内资金投向 |

**关键洞察**：
> 道路来源于背街小巷台账，但申报主题为"道路+管网"，因为中央预算内资金城市更新投向主要支持老旧小区和管网。
