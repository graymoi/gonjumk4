## 1. 概述

### 1.1 政策法规文件夹的作用

政策法规文件夹是精准谋划知识库的核心组成部分，用于存放和管理城乡建设领域的政策文件，为政策咨询、资金申报、项目谋划提供权威依据。

### 1.2 核心价值

- **权威性**：所有文件均为官方发布的正式版本
- **结构化**：按层级、类型、领域、部门等多维度组织
- **可检索**：支持多维度检索和智能匹配
- **时效性**：自动跟踪政策状态，及时更新

### 1.3 主要功能

1. **政策存储**：存放各类政策文件和内部资料
2. **政策检索**：支持多维度检索和智能匹配
3. **政策解读**：提供政策解读和分析文档
4. **政策管理**：跟踪政策状态，管理政策版本
5. **知识图谱**：建立政策之间的关联关系

## 4. 详细操作流程

### 4.1 新增政策文件

#### 步骤1：获取政策文件

**操作方式**：
1. **官方网站获取**
   - 访问各部委官方网站
   - 查找最新发布的政策文件
   - 下载PDF或Word格式文件

2. **政府公报获取**
   - 访问国务院公报网站
   - 查找相关政策文件
   - 下载正式版本

3. **通知公告获取**
   - 关注各部委通知公告
   - 及时获取新发布政策
   - 下载相关文件

**注意事项**：
- 确保下载的是官方正式版本
- 保留文件的原始格式
- 记录文件来源和下载时间

#### 步骤2：文件格式转换

**操作步骤**：
1. **PDF转Markdown**
   - 使用PDF转Markdown工具
   - 保留原文格式和结构
   - 检查转换质量

2. **Word转Markdown**
   - 使用Word转Markdown工具
   - 保留原文格式和结构
   - 检查转换质量

3. **内容整理**
   - 清理多余的格式标记
   - 统一标题层级
   - 优化排版格式

**注意事项**：
- 确保转换后的内容完整准确
- 保留原文的重要格式信息
- 检查特殊字符和表格转换

#### 步骤3：文件命名

**命名规范**：
```
[发文号]_[发文机关]_[政策名称]_[发布日期].md
```

**命名示例**：
- `建科〔2023〕45号_住房城乡建设部_关于推进建设工程质量检测机构资质许可工作的通知_20230510.md`
- `自然资发〔2024〕242号_自然资源部_关于运用地方政府专项债券资金收回收购存量闲置土地的通知_20241101.md`
- `国办发〔2024〕52号_国务院办公厅_关于优化完善地方政府专项债券管理机制的意见_20241225.md`

**命名工具**：
- 使用 `check_policy_naming.py` 检查命名规范
- 使用 `rename_policy_files.py` 自动重命名
- 使用 `batch_rename_all_policies.py` 批量重命名

#### 步骤4：提取元数据

**元数据字段**：

| 字段 | 说明 | 示例 |
|------|------|------|
| file_number | 发文号 | 建科〔2023〕45号 |
| publishing_agency | 发文机关 | 住房城乡建设部 |
| policy_name | 政策名称 | 关于推进建设工程质量检测机构资质许可工作的通知 |
| publishing_date | 发布日期 | 20230510 |
| effective_date | 生效日期 | 20230601 |
| status | 状态 | 现行有效 |
| keywords | 关键词 | 建设工程,质量检测,资质许可 |
| business_domain | 业务领域 | 城市更新,市政基础设施 |
| policy_type | 政策类型 | 规范性文件 |
| level | 层级 | 国家级 |

**元数据模板**：
```yaml
---
file_number: 建科〔2023〕45号
publishing_agency: 住房城乡建设部
policy_name: 关于推进建设工程质量检测机构资质许可工作的通知
publishing_date: 20230510
effective_date: 20230601
status: 现行有效
keywords:
  - 建设工程
  - 质量检测
  - 资质许可
business_domain:
  - 城市更新
  - 市政基础设施
policy_type: 规范性文件
level: 国家级
---
```

#### 步骤5：文件归档

**归档步骤**：
1. **确定归档位置**
   - 按部门归档：`国家级/按部门/[部门名]/`
   - 按领域归档：`国家级/按领域/[领域名]/`

2. **复制文件到目标位置**
   - 将文件复制到相应文件夹
   - 确保文件名符合规范
   - 检查文件内容完整

3. **更新索引文件**
   - 更新部门政策列表
   - 更新领域政策列表
   - 更新总索引文件

**注意事项**：
- 确保归档位置正确
- 检查文件名是否重复
- 及时更新索引文件

### 4.4 政策维护管理

#### 定期检查

**检查频率**：
- **每日**：扫描新政策文件和申报指南
- **每周**：复核新入库文件的关键信息
- **每月**：筛选高频新问题，组织专家解答
- **每季度**：对政策状态、问答有效性进行复核

**检查内容**：
1. **新增政策**
   - 检查是否有新政策发布
   - 下载并入库新政策
   - 更新政策索引

2. **政策修订**
   - 检查是否有政策修订
   - 保留旧版政策
   - 标注修订关系

3. **政策废止**
   - 检查是否有政策废止
   - 标记废止状态
   - 移入已废止目录

#### 版本管理

**管理原则**：
1. **保留历史版本**
   - 重要文件保留历史版本
   - 标注版本关系
   - 记录变更内容

2. **建立版本链**
   - 同一政策有修订版本时，保留旧版
   - 注明替代关系
   - 建立版本链接

3. **年度归档**
   - 每年初将上一年度文件移入"往年"目录
   - 已过期文件移入"已过期"目录
   - 更新索引和元数据

#### 质量控制

**控制措施**：
1. **AI处理抽查**
   - 定期抽查处理结果
   - 验证元数据准确性
   - 检查分类标签

2. **案例入库审核**
   - 脱敏处理个人信息
   - 验证数据真实性
   - 审核案例质量

3. **方法论文件审核**
   - 专家审核重要文件
   - 验证方法有效性
   - 更新版本信息

## 5. 使用场景

### 5.1 政策咨询场景

**用户需求**：用户询问"老旧小区改造有哪些政策支持？"

**操作流程**：
1. **意图识别**
   - 识别问题类型：政策咨询
   - 提取关键词：老旧小区改造
   - 分类领域：城市更新

2. **检索策略**
   - 按领域检索：城市更新
   - 按部门检索：住建部、财政部、发改委
   - 按时间检索：近3年、有效期内

3. **智能检索**
   - 向量检索相关政策
   - 多维过滤结果
   - 相似度排序

4. **结果展示**
   - 政策列表：中央财政补助、专项债券、中央预算内
   - 核心条款：支持范围、申报条件、资金比例
   - 政策对比：不同政策的优劣势、适用场景

5. **推荐建议**
   - 推荐最适合的政策组合
   - 提供申报策略和注意事项

### 5.2 资金申报场景

**用户需求**：用户有一个老旧小区改造项目，需要申报资金

**操作流程**：
1. **项目信息提取**
   - 项目类型：老旧小区改造
   - 投资额：5000万元
   - 建设内容：楼体改造、管网更新、配套设施

2. **特征提取**
   - 项目特征：城市更新、基础设施
   - 领域标签：城市更新、市政基础设施
   - 地域信息：全国/省级/市级

3. **资金匹配**
   - 渠道推荐：中央财政补助、专项债券、中央预算内
   - 条件核查：逐项检查申报条件
   - 匹配度评分：计算各渠道的匹配度

4. **申报准备**
   - 材料清单：可研报告、资金申请报告、项目批复
   - 模板生成：自动生成申报材料模板
   - 撰写指导：提供材料撰写要点

5. **风险评估**
   - 成功率预测：基于历史案例预测成功率
   - 风险识别：识别申报风险因素
   - 改进建议：提供风险防控建议

6. **案例参考**
   - 成功案例：推荐相似成功案例
   - 经验总结：总结成功经验
   - 注意事项：提示申报注意事项

### 5.3 项目谋划场景

**用户需求**：用户计划策划一个城市更新项目

**操作流程**：
1. **项目构思**
   - 项目类型：城市更新项目
   - 建设目标：改善人居环境、提升城市品质
   - 投资估算：2亿元

2. **政策匹配**
   - 适用政策：城市更新中央预算内、专项债券、示范奖补

### C. 常见问题

**Q1：如何快速找到某个部门的政策？**
A：打开 `国家级/按部门/` 目录，选择目标部门文件夹，查看部门政策列表。

**Q2：如何查找某个领域的政策？**
A：打开 `国家级/按领域/` 目录，选择目标领域文件夹，查看领域政策列表。

A：打开 `政策解读/` 目录，选择目标主题文件夹，查看政策解读文档。

A：内部资料统一放在 `国家级/内部资料/` 文件夹中，设置访问控制。
