# 政策图谱视图说明

## 📋 图谱视图说明

本文档说明如何使用 Obsidian 的图谱视图查看政策法规之间的多维联系。

---

## 🎯 图谱视图功能

### 基本功能

| 功能 | 说明 | 使用方法 |
|------|------|---------|
| **节点显示** | 显示所有政策文件 | 打开图谱视图自动显示 |
| **链接显示** | 显示政策之间的链接关系 | 节点之间的连线表示链接 |
| **标签过滤** | 按标签筛选政策 | 点击标签进行筛选 |
| **搜索过滤** | 按关键词搜索政策 | 在搜索框输入关键词 |
| **缩放功能** | 缩放图谱视图 | 滚动鼠标滚轮 |

---

## 🔗 多维联系

### 1. 按部门联系

**显示方式**：
- 相同部门发布的政策会聚集在一起
- 颜色编码表示不同部门

**示例**：
- 发改委政策 → 蓝色节点
- 财政部政策 → 绿色节点
- 住建部政策 → 红色节点

**使用场景**：
- 查看某部门的所有政策
- 了解部门政策体系
- 发现部门政策之间的关联

### 2. 按类型联系

**显示方式**：
- 相同类型的政策会聚集在一起
- 使用标签进行筛选

**示例**：
- 中央预算内投资 → `#中央预算内投资`
- 专项债券 → `#专项债券`
- REITs → `#REITs`

**使用场景**：
- 查看某类型的所有政策
- 了解政策类型体系
- 发现类型政策之间的关联

### 3. 按时间联系

**显示方式**：
- 使用发布时间进行排序
- 新政策在图谱中更突出

**示例**：
- 2026年政策 → 最新
- 2025年政策 → 次新
- 2024年政策 → 较早

**使用场景**：
- 查看最新政策
- 了解政策演进
- 发现政策更新

### 4. 按关联联系

**显示方式**：
- 相关政策之间有连线
- 连线粗细表示关联强度

**示例**：
- 上位政策 → 下位政策
- 配套政策 → 实施细则
- 相关政策 → 交叉引用

**使用场景**：
- 查看政策体系
- 了解政策层级
- 发现政策关联

## 🔍 使用技巧

### 技巧1：按标签筛选

**步骤**：
1. 打开图谱视图
2. 点击标签面板
3. 选择要筛选的标签
4. 图谱只显示带该标签的政策

**示例**：
- 筛选 `#中央预算内投资` → 只显示中央预算内投资政策
- 筛选 `#城市更新` → 只显示城市更新政策

### 技巧2：按部门筛选

**步骤**：
1. 打开图谱视图
2. 在搜索框输入部门名称
3. 图谱只显示该部门的政策

**示例**：
- 搜索 "发改委" → 只显示发改委政策
- 搜索 "财政部" → 只显示财政部政策

### 技巧3：查看政策关联

**步骤**：
1. 在图谱中点击某个政策节点
2. 查看与该政策相连的节点
3. 点击连线查看链接详情

**示例**：
- 点击 "节能降碳专项管理办法" → 查看相关政策
- 点击 "城市更新示范工作通知" → 查看相关政策

### 技巧4：发现政策体系

**步骤**：
1. 打开图谱视图
2. 查看聚集在一起的节点群
3. 分析节点群之间的关系

**示例**：
- 中央预算内投资节点群 → 了解中央预算内投资政策体系
- 城市更新节点群 → 了解城市更新政策体系

## 📊 图谱视图应用

### 应用1：政策研究

**场景**：研究某领域政策体系

**方法**：
1. 按类型筛选该领域政策
2. 查看政策之间的关联
3. 分析政策体系结构

**示例**：
- 研究中央预算内投资政策体系
- 研究城市更新政策体系
- 研究专项债券政策体系

### 应用2：项目谋划

**场景**：为项目谋划寻找政策支持

**方法**：
1. 按项目类型筛选相关政策
2. 查看政策之间的关联
3. 选择最适合的政策组合

**示例**：
- 老旧小区改造项目 → 查看城市更新、保障性住房政策
- 基础设施项目 → 查看专项债券、中央预算内投资政策

**场景**：回答政策咨询问题

**方法**：
1. 按问题关键词搜索政策
2. 查看政策之间的关联
3. 提供全面的政策解答

**示例**：
- 专项债券咨询 → 查看专项债券相关政策
- 中央预算内投资咨询 → 查看中央预算内投资相关政策

## 🎯 最佳实践

### 1. 定期更新图谱

**频率**：每周更新一次

**内容**：
- 添加新政策
- 更新政策链接
- 删除废止政策

### 2. 保持链接准确

**原则**：
- 只建立真实的政策关联
- 避免过度链接
- 定期检查链接有效性

### 3. 使用标准标签

**规范**：
- 使用标准标签体系
- 避免自定义标签
- 定期清理无用标签
