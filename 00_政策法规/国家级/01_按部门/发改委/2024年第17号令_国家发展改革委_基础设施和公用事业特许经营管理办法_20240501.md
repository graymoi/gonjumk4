# 基础设施和公用事业特许经营管理办法（2024年第17号令）

---

## 一、总则

### 第一条【立法目的】

为鼓励和引导社会资本参与基础设施和公用事业建设运营，提高公共服务质量和效率，保护特许经营者合法权益，激发民间投资活力，保障社会公共利益和公共安全，促进经济社会持续健康发展，制定本办法。

### 第二条【适用范围】

中华人民共和国境内的交通运输、市政工程、生态保护、环境治理、水利、能源、体育、旅游等基础设施和公用事业领域的特许经营活动，适用本办法。

### 第三条【特许经营定义】

基础设施和公用事业特许经营，是指政府采用公开竞争方式依法选择中华人民共和国境内外的法人或者其他组织作为特许经营者，通过协议明确权利义务和风险分担，约定其在一定期限和范围内投资建设运营基础设施和公用事业并获得收益，提供公共产品或者公共服务。

**排除情形：**
商业特许经营以及不涉及产权移交环节的公建民营、公办民营等，不属于本办法所称基础设施和公用事业特许经营。

### 第四条【PPP新机制】

基础设施和公用事业特许经营是基于使用者付费的政府和社会资本合作（PPP）模式，政府就项目投资建设运营与社会资本开展合作，不新设行政许可。

**核心特征：**
1. 特许经营者获得协议约定期限内对特定基础设施和公用事业项目进行投资建设运营并取得收益的排他性权利
2. 应当按照协议约定提供符合质量效率要求的公共产品或者公共服务
3. 依法接受监督管理

**禁止行为：**
- 禁止无法律法规依据擅自增设行政许可事项以及通过前述擅自增设的行政许可事项向特许经营者收费，增加公共产品或者公共服务成本
- 禁止以基础设施和公用事业特许经营名义滥用行政权力排除、限制竞争

### 第五条【基本原则】

基础设施和公用事业特许经营应当坚持公开、公平、公正，保护各方信赖利益，并遵循以下原则：

1. **发挥社会资本优势**：发挥社会资本融资、专业、技术和管理优势，提高公共服务质量效率
2. **转变政府职能**：强化政府与社会资本协商合作
3. **保护合法权益**：保护社会资本合法权益，保证特许经营持续性和稳定性
4. **兼顾公益平衡**：兼顾经营性和公益性平衡，维护公共利益
5. **明确风险分担**：根据风险性质以及特许经营各方风险管控能力，明确风险分担机制并切实执行，保障项目持续稳定实施

### 第六条【使用者付费】

基础设施和公用事业特许经营应当聚焦使用者付费项目，明确收费渠道和方式，项目经营收入具备覆盖建设投资和运营成本并获取一定投资回报的条件，不因采取特许经营而额外新增地方财政未来支出责任。

**使用者付费定义：**
包括特许经营者直接向用户收费，以及由政府或其依法授权机构代为向用户收费。

### 第七条【特许经营方式】

基础设施和公用事业特许经营可以采取以下方式：

1. **新建-运营-移交（BOT）**：在一定期限内，政府授予特许经营者投资新建或改扩建、运营基础设施和公用事业，期限届满移交政府
2. **新建-拥有-运营-移交（BOOT）**：在一定期限内，政府授予特许经营者投资新建或改扩建、拥有并运营基础设施和公用事业，期限届满移交政府
3. **转让-运营-移交（TOT）**：在一定期限内，政府将已经建成的基础设施和公用事业项目转让特许经营者运营，期限届满移交政府
4. **其他方式**：国家规定的其他方式

**禁止行为：**
禁止在建设工程完成后直接将项目移交政府，或者通过提前终止协议等方式变相逃避运营义务。

### 第八条【特许经营期限】

基础设施和公用事业特许经营期限应当根据行业特点、所提供公共产品或服务需求、项目生命周期、项目建设投资和运营成本、投资回收期等综合因素确定，充分保障特许经营者合法权益。

**期限规定：**
特许经营期限原则上不超过40年，投资规模大、回报周期长的特许经营项目可以根据实际情况适当延长，法律法规另有规定的除外。

### 第九条【部门职责】

**国务院发展改革部门：**
牵头做好基础设施和公用事业特许经营模式推进工作，加强政策指导

**地方各级发展改革部门：**
发挥综合协调作用，严格把关项目实施领域、范围、方案等，依法依规履行项目审批、核准或者备案职责，推动项目落地实施

**国务院有关行业主管部门：**
应当根据国务院关于规范实施政府和社会资本合作新机制的要求，按照各自职责履行行业管理职能

**政策协调机制：**
涉基础设施和公用事业特许经营政策事项，应当送国务院发展改革部门充分征求意见并经评估与宏观政策取向一致，未经评估论证或者经评估论证与宏观政策取向不一致的，不应当公布实施

## 二、特许经营项目的实施

### 第十条【项目前期工作】

特许经营项目应当加强前期研究论证，包括：

1. 项目可行性研究
2. 特许经营方案编制
3. 财务评估
4. 法律意见
5. 风险评估

### 第十一条【特许经营方案】

特许经营方案应当包括以下内容：

1. 项目概况
2. 特许经营方式
3. 特许经营期限
4. 特许经营者选择方式
5. 投资回报机制
6. 风险分担机制
7. 监管要求
8. 移交标准

### 第十二条【特许经营者选择】

特许经营者应当通过公开竞争方式选择，包括：

1. 招标
2. 竞争性谈判
3. 竞争性磋商
4. 其他公开竞争方式

**民营企业优先：**
鼓励民营企业通过直接投资、独资、控股、参与联合体等多种方式参与特许经营项目

### 第十三条【特许经营协议】

特许经营协议应当明确以下内容：

1. 项目名称、地点、内容、规模、标准
2. 特许经营方式、期限
3. 投资规模、出资方式
4. 收费标准、调整机制
5. 服务质量标准
6. 风险分担机制
7. 监管要求
8. 违约责任
9. 争议解决方式
10. 移交标准、程序

### 第十四条【项目实施】

特许经营者应当按照特许经营协议约定，投资建设运营基础设施和公用事业项目，提供符合质量效率要求的公共产品或者公共服务。

### 第十五条【项目监管】

政府应当加强对特许经营项目的监管，包括：

1. 建设监管
2. 运营监管
3. 服务质量监管
4. 价格监管
5. 安全监管

---

## 三、特许经营者的权利和义务

### 第十六条【特许经营者权利】

特许经营者享有以下权利：

1. 按照特许经营协议约定获得收益
2. 在特许经营期限内对项目进行投资建设运营
3. 要求政府按照特许经营协议约定履行义务
4. 在特许经营期限届满时获得合理补偿

### 第十七条【特许经营者义务】

特许经营者应当履行以下义务：

1. 按照特许经营协议约定投资建设运营项目
2. 提供符合质量效率要求的公共产品或者公共服务
3. 接受政府监管
4. 承担特许经营协议约定的风险
5. 在特许经营期限届满时按照约定移交项目

---

## 四、特许经营协议的变更、终止和移交

### 第十八条【协议变更】

特许经营协议的变更应当符合以下条件：

1. 经特许经营双方协商一致
2. 不损害社会公共利益
3. 不影响特许经营项目的正常运营

### 第十九条【协议终止】

特许经营协议的终止应当符合以下情形：

1. 特许经营期限届满
2. 特许经营双方协商一致
3. 特许经营者严重违约
4. 不可抗力
5. 法律法规规定的其他情形

### 第二十条【项目移交】

特许经营期限届满，特许经营者应当按照特许经营协议约定将项目移交给政府，移交应当符合以下要求：

1. 项目处于良好运营状态
2. 符合约定的技术标准
3. 相关资料完整齐全
4. 人员培训到位

---

## 五、监督管理

### 第二十一条【政府监管】

政府应当加强对特许经营项目的监督管理，包括：

1. 建立健全监管制度
2. 加强日常监管
3. 开展定期检查
4. 建立信息披露制度

### 第二十二条【绩效评价】

政府应当对特许经营项目进行绩效评价，评价内容包括：

1. 项目建设情况
2. 运营情况
3. 服务质量
4. 投资回报
5. 社会效益

### 第二十三条【风险防控】

政府应当建立健全风险防控机制，包括：

1. 建立风险评估机制
2. 建立风险预警机制
3. 建立风险处置机制
4. 建立风险分担机制

---

## 六、法律责任

### 第二十四条【政府责任】

政府及其工作人员有下列情形之一的，依法给予处分；构成犯罪的，依法追究刑事责任：

1. 违反规定选择特许经营者
2. 违反规定签订特许经营协议
3. 不履行或者不正确履行监管职责
4. 滥用职权、玩忽职守、徇私舞弊

### 第二十五条【特许经营者责任】

特许经营者有下列情形之一的，依法承担法律责任：

1. 不履行或者不正确履行特许经营协议
2. 提供不符合质量效率要求的公共产品或者公共服务
3. 拒绝接受政府监管
4. 违反法律法规规定

## 七、附则

### 第二十六条【施行时间】

本办法自2024年5月1日起施行。

### 第二十七条【解释权】

本办法由国家发展改革委负责解释。

## 八、政策意义

### （一）建立PPP新机制

明确特许经营是基于使用者付费的PPP模式，统一了特许经营与PPP的内涵，开启了"使用者付费"的PPP新机制。

### （二）鼓励民营企业参与

鼓励民营企业通过直接投资、独资、控股、参与联合体等多种方式参与特许经营项目，解决民营企业入场难问题。

### （三）规范项目管理

改进特许经营项目管理程序，强化特许经营项目前期研究论证，加强事中事后监管，强调投资者权益保护。

### （四）防范隐性债务

聚焦使用者付费项目，不因采取特许经营而额外新增地方财政未来支出责任，防范隐性债务风险。
