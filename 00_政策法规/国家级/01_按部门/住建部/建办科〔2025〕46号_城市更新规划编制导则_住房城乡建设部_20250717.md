# 建办科〔2025〕46号 - 城市更新规划编制导则

---

各省、自治区住房城乡建设厅，直辖市住房城乡建设（管）委，新疆生产建设兵团住房城乡建设局：

为深入贯彻中央城市工作会议精神，落实《中共中央办公厅、国务院办公厅关于持续推进城市更新行动的意见》，指导各地制定实施城市更新专项规划，建立完善"专项规划—片区策划—项目实施方案"的规划实施体系，住房城乡建设部组织编制了《城市更新规划编制导则》，现印发给你们，请结合实际做好城市更新规划编制实施工作。

## 第一章 总则

### 第一条【编制目的】
为规范城市更新规划编制工作，提高规划编制质量，发挥规划引领作用，有力有序有效实施城市更新重点任务，制定本导则。

### 第二条【适用范围】
本导则适用于全国设市城市和县人民政府所在地的镇的城市更新规划编制工作。

### 第三条【规划体系】
城市更新规划体系包括：
1. 城市更新专项规划
2. 片区策划
3. 项目实施方案

### 第四条【编制原则】
城市更新规划编制应当遵循以下原则：
1. 以人为本、民生优先
2. 传承文脉、留住乡愁
3. 绿色低碳、智慧高效
4. 量力而行、尽力而为
5. 多元参与、共建共享

## 第二章 城市更新专项规划

### 第五条【规划定位】
城市更新专项规划是国土空间规划体系中的专项规划，是指导城市更新工作的纲领性文件。

### 第六条【规划期限】
城市更新专项规划期限应当与国土空间总体规划期限相衔接，一般为5年。

### 第七条【规划内容】
城市更新专项规划应当包括以下内容：
1. 城市更新现状评估
2. 城市更新总体目标
3. 城市更新空间布局
4. 城市更新重点任务
5. 城市更新实施时序
6. 城市更新保障措施

### 第八条【现状评估】
城市更新现状评估应当包括：
1. 城市建设现状分析
2. 城市功能布局分析
3. 城市历史文化资源分析
4. 城市基础设施分析
5. 城市公共服务设施分析
6. 城市更新需求分析

### 第九条【空间布局】
城市更新空间布局应当：
1. 划定城市更新单元
2. 确定城市更新重点区域
3. 明确城市更新时序安排
4. 统筹城市更新与国土空间规划

### 第十条【重点任务】
城市更新重点任务应当包括：
1. 老旧小区改造
2. 城中村改造
3. 老旧街区更新
4. 老旧厂区改造
5. 城市基础设施更新
6. 城市公共服务设施完善

## 第三章 片区策划

### 第十一条【策划定位】
片区策划是城市更新专项规划与项目实施方案之间的桥梁，是指导片区更新的重要依据。

### 第十二条【策划范围】
片区策划范围应当根据城市更新单元划定，原则上以街道、社区为基本单元。

### 第十三条【策划内容】
片区策划应当包括：
1. 片区现状分析
2. 片区更新目标
3. 片区更新策略
4. 片区功能布局
5. 片区更新时序
6. 片区资金平衡方案

### 第十四条【功能布局】
片区功能布局应当：
1. 优化片区功能结构
2. 完善片区公共服务设施
3. 改善片区交通组织
4. 提升片区环境品质

### 第十五条【资金平衡】
片区资金平衡方案应当：
1. 测算片区更新投入
2. 预测片区更新收益
3. 明确资金筹措渠道
4. 制定资金使用计划

## 第四章 项目实施方案

### 第十六条【方案定位】
项目实施方案是实施城市更新项目的具体操作指南。

### 第十七条【方案内容】
项目实施方案应当包括：
1. 项目基本情况
2. 项目更新目标
3. 项目更新内容
4. 项目实施计划
5. 项目资金筹措方案
6. 项目运营管理方案

### 第十八条【实施计划】
项目实施计划应当明确：
1. 项目实施时序
2. 项目实施主体
3. 项目实施责任
4. 项目实施保障措施

### 第十九条【资金筹措】
项目资金筹措方案应当包括：
1. 资金需求测算
2. 资金来源渠道
3. 资金使用计划
4. 资金保障措施

## 第五章 规划实施保障

### 第二十条【组织保障】
建立城市更新工作协调机制，明确各部门职责分工，形成工作合力。

### 第二十一条【政策保障】
完善城市更新政策体系，制定配套政策，为城市更新提供政策支持。

### 第二十二条【资金保障】
建立多元化资金筹措机制，保障城市更新资金需求。

### 第二十三条【技术保障】
加强城市更新技术支撑，推广应用新技术、新工艺、新材料。

### 第二十四条【监督保障】
建立城市更新监督机制，加强对城市更新规划实施的监督检查。

## 第六章 附则

### 第二十五条【解释权】
本导则由住房城乡建设部负责解释。

### 第二十六条【施行日期】
本导则自印发之日起施行。
