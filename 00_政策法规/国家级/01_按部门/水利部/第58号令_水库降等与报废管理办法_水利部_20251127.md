# 水库降等与报废管理办法

## 基本信息

- **文号**：中华人民共和国水利部令第58号
- **发布机关**：水利部
- **发布日期**：2025年11月27日
- **文件类型**：部门规章
- **发布来源**：国务院公报2025年第33号

## 主要内容

### 适用范围
本办法适用于中华人民共和国境内水库的降等与报废管理。

### 降等与报废条件
1. **降等条件**：水库因工程状况、功能变化等原因需要降低等级
2. **报废条件**：水库因工程老化、功能丧失等原因需要报废

### 核心要点
1. **降等程序**：明确水库降等的申请、审批、实施程序
2. **报废程序**：明确水库报废的申请、审批、实施程序
3. **安全管理**：加强降等与报废过程中的安全管理
4. **责任追究**：明确违反本办法的法律责任

## 核心要点

### 1. 降等管理
- **降等申请**：水库管理单位提出降等申请
- **技术评估**：组织技术评估论证
- **审批程序**：按照权限分级审批
- **实施要求**：制定降等实施方案

### 2. 报废管理
- **报废申请**：水库管理单位提出报废申请
- **安全评估**：进行安全风险评估
- **审批程序**：按照权限分级审批
- **实施要求**：制定报废实施方案

### 3. 安全管理
- **安全监测**：加强降等与报废过程的安全监测
- **应急预案**：制定应急预案
- **安全措施**：落实安全防护措施
- **人员疏散**：必要时组织人员疏散

### 4. 资产处置
- **资产清查**：进行资产清查登记
- **处置程序**：按照规定程序处置资产
- **资金管理**：规范资金使用管理
- **档案管理**：完善档案资料管理

## 重要意义

### 1. 保障水利工程安全
- 规范水库降等与报废管理
- 消除水利工程安全隐患
- 保障人民生命财产安全

### 2. 优化资源配置
- 合理调整水库等级
- 优化水资源配置
- 提高水资源利用效率

### 3. 规范管理程序
- 明确降等与报废程序
- 规范审批流程
- 提高管理水平

## 对城乡建设领域的影响

### 1. 水利设施建设
- 促进水利设施标准化建设
- 推动水利设施现代化改造
- 提升水利设施管理水平

### 2. 城市防洪排涝
- 保障城市防洪安全
- 提升城市排涝能力
- 支持城市更新行动

### 3. 乡村振兴
- 支持农村水利设施建设
- 促进农村水利设施改造
- 推动乡村振兴战略实施

### 4. 生态环境保护
- 促进生态环境修复
- 保护水生态环境
- 推动生态文明建设
