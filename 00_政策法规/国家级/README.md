## 使用说明

### 1. 按部门查找

**适用场景**：知道发文机关，需要查找该部门的所有政策

**操作方法**：
1. 进入【按部门】目录
2. 选择对应部门文件夹
3. 浏览该部门的所有政策文件

**示例**：
- 查找发改委的政策 → `按部门/发改委/`
- 查找财政部的政策 → `按部门/财政部/`

---

### 2. 按领域查找

**适用场景**：需要查找特定领域的所有政策，不限定发文机关

**操作方法**：
1. 进入【按领域】目录
2. 打开对应领域的索引文件
3. 在索引中找到需要的政策
4. 点击链接跳转到实际文件

**示例**：
- 查找专项债券政策 → `按领域/专项债券索引.md`
- 查找中央预算内资金政策 → `按领域/中央预算内资金索引.md`
- 查找城市更新政策 → `按领域/城市更新索引.md`

---

### 3. 跨目录跳转

**实现方式**：通过索引文件中的相对路径链接

**跳转流程**：
```
按领域/专项债券索引.md
    ↓ 点击链接
按部门/国务院办公厅/国办发〔2024〕52号_国务院办公厅_关于优化完善地方政府专项债券管理机制的意见_20241225.md
```

## 维护说明

### 新增政策

**步骤**：
1. 将政策文件添加到【按部门】对应部门目录
2. 更新相关领域的索引文件
3. 如果是法律或行政法规，更新对应索引

### 修改政策

**步骤**：
1. 修改【按部门】目录下的政策文件
2. 如果政策名称或发文机关变化，更新相关索引

### 删除政策

**步骤**：
1. 删除【按部门】目录下的政策文件
2. 更新相关领域的索引文件
3. 从索引中移除对应条目

## 优化效果

### 存储优化
- **优化前**：政策文件在多个目录重复存储
- **优化后**：政策文件只在【按部门】存储一次
- **效果**：减少存储空间，避免版本不一致

### 查找优化
- **按部门查找**：保持原有方式，直接进入部门目录
- **按领域查找**：通过索引快速定位，支持跨部门查找
- **效果**：提高查找效率，支持多维度查找

### 维护优化
- **优化前**：修改政策需要更新多个目录
- **优化后**：修改政策只需更新一个文件
- **效果**：减少维护工作量，避免遗漏

## 注意事项

### 文件存储原则
- **唯一存储**：政策文件只在【按部门】目录存储
- **索引跳转**：其他目录通过索引跳转到【按部门】
- **内部资料**：内部资料保留在【内部资料】目录

### 索引维护原则
- **及时更新**：新增、修改、删除政策时及时更新索引
- **链接检查**：定期检查索引链接是否有效
- **统计更新**：定期更新索引统计信息

### 跨目录跳转原则
- **相对路径**：使用相对路径，便于目录移动
- **清晰标注**：在索引中标注文件存储位置
- **快速定位**：确保链接准确，支持快速跳转
