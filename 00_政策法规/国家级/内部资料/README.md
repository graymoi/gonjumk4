# 内部资料文件夹

## 说明

本文件夹用于存储内部政策资料，包括未发布文件、征求意见稿、会议纪要等非公开内容。

## 文件类型

### 1. 未发布文件
- 征求意见稿
- 草案
- 待发布文件

### 2. 内部文件
- 会议纪要
- 内部通知
- 工作文件
- 研究报告

## 命名规范

### 未发布文件
- 前缀：`[未发布]`
- 示例：`[未发布]2025年城市更新专项资金申报指南（征求意见稿）.md`

### 内部文件
- 前缀：`[内部]`
- 示例：`[内部]关于XX片区规划的会议纪要.md`

## 元数据规范

内部文件必须包含以下元数据：

```yaml
---
title: 文件标题
file_number: 文号（如有）
publishing_agency: 发布部门
publishing_date: 发布日期
document_type: 文件类型
classification: 内部  # 必须标记为内部
effectiveness: 有效/未发布/征求意见稿
---
```

## 访问控制

- 本文件夹应设置严格的访问控制
- 建议加密存储
- 仅限内网访问
- 记录访问日志

## AI检索说明

当AI进行检索时，会根据用户权限过滤结果：
- 即使文件在内部文件夹，也需确保用户有权访问
- 内部文件不会向未授权用户展示

## 文件示例

```
内部资料/
├── [未发布]2025年城市更新专项资金申报指南（征求意见稿）.md
├── [未发布]关于XX政策的草案.md
├── [内部]关于XX片区规划的会议纪要.md
└── [内部]XX项目研究报告.md
```
