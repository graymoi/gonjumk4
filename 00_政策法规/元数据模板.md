# 政策文件元数据模板

## 📋 元数据字段说明

本模板定义了政策文件的标准元数据格式，用于建立政策知识库的多维联系。

---

## 🎯 必填字段

### 基础信息

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `title` | string | 政策标题 | 发改环资规〔2025〕1228号 - 节能降碳中央预算内投资专项管理办法 |
| `file_number` | string | 发文字号 | 发改环资规〔2025〕1228号 |
| `publishing_agency` | string | 发布机构 | 国家发展改革委 |
| `publishing_date` | date | 发布日期 | 2025-09-19 |
| `document_type` | string | 文件类型 | 专项管理办法 |
| `level` | string | 政策级别 | 部委级/省级/市级/县级 |
| `effectiveness` | string | 效力状态 | 有效/废止/修订中 |

### 分类信息

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `category` | string | 政策领域 | 中央预算内投资/专项债券/REITs/城市更新/乡村振兴 |
| `tags` | list | 标签列表 | ["中央预算内投资", "节能降碳", "碳达峰碳中和", "绿色转型"] |

---

## 🔗 可选字段

### 关联信息

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `supervising_agency` | string | 监管机构 | 国家发展改革委 |
| `online_url` | string | 官方链接 | https://www.gov.cn/... |
| `related_policies` | list | 相关政策 | ["发改环资规〔2024〕338号", "政府投资条例"] |
| `knowledge_links` | list | 知识链接 | [["政策知识链路和关联关系"], ["知识图谱_资金政策"]] |

### 时间信息

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `effective_date` | date | 生效日期 | 2025-10-01 |
| `expiry_date` | date | 失效日期 | 2030-12-31 |
| `created_date` | date | 入库日期 | 2025-02-20 |
| `updated_date` | date | 更新日期 | 2025-02-20 |

### 业务信息

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `funding_type` | string | 资金类型 | 中央预算内投资/专项债券/财政补助/贴息 |
| `project_type` | list | 适用项目类型 | ["节能改造", "清洁能源", "循环经济"] |
| `region` | string | 适用区域 | 全国/东部地区/中部地区/西部地区 |
| `amount_limit` | string | 金额限制 | 单个项目不超过5000万元 |

## 🔗 链接关系

### 相关政策链接

使用 `related_policies` 字段建立政策之间的关联关系：

```yaml
related_policies:
  - 发改环资规〔2024〕338号
  - 政府投资条例
  - 中央预算内投资项目监督管理办法
```

### 知识链接

使用 `knowledge_links` 字段建立与知识库其他部分的链接：

```yaml
knowledge_links:
  - [[政策知识链路和关联关系]]
  - [[知识图谱_资金政策]]
  - [[政策文件汇总_知识链接]]
```

## 🎯 使用建议

1. **必填字段**：所有政策文件必须填写必填字段
2. **标签规范**：使用标准标签，避免自定义标签
3. **链接关系**：建立政策之间的关联关系，便于知识图谱展示
4. **定期更新**：定期更新元数据，确保信息准确
