# 基础设施REITs首单发行案例

## 一、水利基础设施REITs

### 1.1 银华绍兴原水REIT

**基本信息：**
- 产品名称：银华绍兴原水封闭式基础设施证券投资基金
- 上市时间：2024年11月8日
- 交易所：深圳证券交易所
- 资产类型：水利基础设施
- 项目性质：全国首单水利基础设施REITs

**项目意义：**
- 推动REITs市场扩容增类
- 深化水利领域投融资体制机制改革
- 探索水利基础设施盘活路径
- 为水利基础设施建设提供新思路

**市场影响：**
- 开创水利基础设施资产证券化先河
- 为水利项目提供退出渠道
- 促进水利基础设施市场化运营
- 提高水利项目投资效率

## 二、仓储物流REITs

### 2.1 华安外高桥仓储物流REIT

**基本信息：**
- 产品名称：华安外高桥仓储物流封闭式基础设施证券投资基金
- 发售时间：2024年12月6日
- 认购价格：2.790元/份
- 计划上市时间：2024年12月25日
- 交易所：上海证券交易所
- 项目性质：上海首单仓储物流基础设施REITs

**原始权益人：**
- 上海外高桥集团股份有限公司

**基金管理人：**
- 华安基金管理有限公司

**财务顾问：**
- 国泰君安证券股份有限公司

**项目特点：**
- 上海自贸区核心区域仓储物流资产
- 具有稳定的租赁收入
- 符合国家物流基础设施发展方向
- 具有良好的市场前景

### 2.2 汇添富九州通医药REIT

**基本信息：**
- 产品名称：汇添富九州通医药仓储物流封闭式基础设施证券投资基金
- 批准时间：2024年12月24日
- 基金期限：39年
- 拟集份额总额：4亿份
- 交易所：上海证券交易所
- 项目性质：国内医药仓储物流设施首单公募REITs、湖北省首单民营企业公募REITs

**底层资产：**
- 武汉东西湖现代医药物流仓储项目
- 地理位置：武汉市东西湖区
- 地理优势：覆盖周边30公里范围内的多家三甲医院
- 功能定位：医药物流保供

**原始权益人：**
- 九州通（600998）

**项目意义：**
- 首单医药仓储物流基础设施公募REITs
- 湖北省首单民营企业公募REITs
- 具有较强的示范效应和积极影响
- 促进医药仓储物流基础设施发展

**市场表现：**
- 2024年前三季度收入1134.29亿元
- 归母净利润16.96亿元
- 具有良好的经营基础

## 三、市政基础设施REITs

**基本信息：**
- 产品名称：国泰君安济南能源供热封闭式基础设施证券投资基金

- 交易所：上海证券交易所
- 项目性质：全国首单市政基础设施公募REITs、全国首单供热基础设施公募REITs

**底层资产：**
- 济南市历下区、历城区的232段优质供热管网资产
- 覆盖面积：3700.20万平方米
- 服务用户：37.45万户
- 服务区域：济南核心功能区

**原始权益人：**
- 济南热力集团

**基金管理人：**
- 国泰君安资管

**财务顾问：**
- 国泰君安证券

**项目特点：**
- 聚焦市政供热优质资产
- 依托民生刚需稳健经营
- 资产运营较为成熟稳定
- 区域经济发展良好、常住人口持续增加

**项目意义：**
- 民生保障与金融创新的重要融合实践
- 在盘活市政基础设施资产方面形成良好示范效应
- 保障居民生活质量、提升城市服务功能
- 开拓创新公募REITs大类资产类型

## 六、2025年REITs市场发展情况

### 6.1 市场规模

**总体情况：**
- 2025年REITs产品新增发行20单
- 2024年为29单、2023年为5单
- 2022年为11单、2021年为13单
- 市场规模持续扩大

**市场表现：**
- 中证REITs（收盘）指数：839.97
- 同比上涨3.50%
- 2024年年初至今累计上涨11.04%
- 中证REITs全收益指数：1033.17
- 同比上涨11.47%
- 2024年年初至今累计上涨19.89%

### 6.2 资产类型分布

**主流资产类型：**
- 产业园区
- 收费公路
- 仓储物流
- 清洁能源
- 保障性租赁住房
- 消费设施
- 水利设施
- 生态环保
- 交通基础设施
- 供热基础设施
- 数据中心

**深交所上市情况：**
- 截至2025年1月3日，共有20只REITs在深交所上市
- 募集规模合计530.06亿元
- 2024年11只REITs在深交所实现平稳上市
- 上市项目数量较年初实现翻一番
- 发行规模超过去三年之和

### 6.3 首单项目汇总

**2025年首单项目：**
1. 首单数据中心公募REITs（南方万国数据中心REIT）
2. 首单城市更新产业园基础设施公募REITs（华夏金隅智造工场REIT）
3. 首单外资消费基础设施公募REITs（华夏凯德商业REIT）
4. 首单水利基础设施公募REITs（银华绍兴原水REIT）
5. 首单市政基础设施公募REITs（国泰君安济南能源供热REIT）
6. 首单供热基础设施公募REITs（国泰君安济南能源供热REIT）
7. 首单医药仓储物流基础设施公募REITs（汇添富九州通医药REIT）
8. 上海首单仓储物流基础设施REITs（华安外高桥仓储物流REIT）

## 七、政策支持

### 7.1 政策导向

**国家发展改革委：**
- 支持符合条件的民间投资项目发行基础设施REITs
- 推动市场扩围扩容
- 指导各地建立健全民间投资项目发行基础设施REITs专项协调服务机制
- 加快推动解决民间投资项目培育中的困难和问题
- 用好基础设施REITs项目信息系统

**中国证监会：**
- 优化申报推荐流程
- 动态完善有关项目申报要求
- 提高工作质效
- 在严防风险、严把质量的基础上，支持更多符合条件的项目发行上市

### 7.2 配套措施

**简化发行流程：**
- 优化审批程序
- 缩短审批时间
- 提高审批效率

**优化估值与交易体系：**
- 完善估值方法
- 优化交易机制
- 提高市场流动性

**扩大长期资金参与：**
- 鼓励保险资金、养老金等长期资金投资
- 提高市场稳定性
- 促进市场健康发展

**推进专项立法：**
- 完善法律法规
- 明确各方权利义务
- 保障市场健康发展

## 八、市场前景

### 8.1 发展趋势

**市场扩容：**
- 资产类型持续增加
- 市场规模不断扩大
- 投资者参与度提高
- 市场功能逐步发挥

**资产多元化：**
- 传统基础设施资产
- 新型基础设施资产
- 民生保障资产
- 城市更新资产

### 8.2 投资机会

**投资者关注点：**
- 消费基础设施、租赁住房、数据中心成为投资者最关注的三大资产类型
- 长租公寓/服务式公寓凭借需求稳定及政策支持成为投资热点
- 零售物业关注度显著回升，购物中心、奥特莱斯、社区商业成为核心细分领域

**投资策略：**
- 险资与不动产私募基金为保障性租赁住房投资主力
- 险资偏好具备稳定现金流的资产
- 不动产私募基金聚焦"商改租"等增值型策略

## 九、风险提示

### 9.1 政策风险

- 政策变化影响
- 审批标准调整
- 监管要求变化
- 税收政策调整

### 9.2 市场风险

- 市场波动风险
- 流动性风险
- 利率风险
- 通胀风险

### 9.3 项目风险

- 运营风险
- 资产质量风险
- 现金流风险
- 管理风险

## 十、总结

基础设施REITs市场在2024-2025年取得了显著发展，多个首单项目成功发行，资产类型不断扩容，市场规模持续扩大。政策支持力度持续加大，市场机制不断完善，投资者认可度不断提高。未来，随着政策支持力度加大和市场机制完善，基础设施REITs将迎来更大发展空间，为基础设施建设和存量资产盘活提供重要支持。

**关键要点：**
1. 多个首单项目成功发行，市场影响力显著
2. 资产类型持续扩容，覆盖范围不断扩大
3. 政策支持力度持续加大，配套措施不断完善
4. 市场规模持续扩大，投资者参与度提高
5. 需要关注政策风险、市场风险和项目风险
