# 政策解读

本目录存放对政策文件的解读性文档，这些文档基于政策原文进行分析、解读和说明，为政策理解和应用提供参考。

## 文件夹结构

```
政策解读/
├── 城市更新/          # 城市更新相关政策解读
├── REITs/             # 基础设施REITs相关政策解读
├── 专项债券/           # 地方政府专项债券相关政策解读
├── 财政金融/           # 财政金融相关政策解读
├── 市政基础设施/       # 市政基础设施相关政策解读
└── 保障性安居工程/     # 保障性安居工程相关政策解读
```

## 文档特点

政策解读文档具有以下特点：

1. **基于政策原文**：所有解读都基于正式发布的政策文件，确保准确性
2. **非原文性质**：这些文档不是政策原文，而是对政策的解读和分析
3. **需要原文支撑**：解读内容需要引用政策原文作为依据
4. **实用导向**：解读注重政策应用和实际操作

## 与政策原文的关系

- **政策原文**：存放在 `00_政策法规/国家级/按部门/` 和 `00_政策法规/国家级/按领域/` 目录下
- **政策解读**：存放在本目录下，是对政策原文的补充说明

## 使用建议

1. 先阅读政策原文，了解政策的基本内容和要求
2. 再查阅政策解读，深入理解政策的具体应用和操作要点
3. 结合实际工作，将政策解读内容应用到具体项目中

## 注意事项

- 政策解读仅供参考，具体政策执行以官方发布的政策原文为准
- 如有疑问，应以政策原文或官方解释为准
- 政策解读会根据政策更新及时修订
